# Security Policy

## Supported Versions

This project is currently in active development. Only the latest release
and the main branch are supported for security fixes.

## Reporting a Vulnerability

Please report security issues privately. Do not open a public issue.

Preferred method:

- Open a GitHub Security Advisory for this repository.

If that is not possible, contact the maintainers at:

- eran@aethernex.com

Please include:

- A detailed description of the issue
- Steps to reproduce
- Impact assessment
- Any relevant logs, proofs, or patches

## Response Targets

- Acknowledge receipt within 72 hours
- Provide an initial assessment within 7 days
- Release a fix as soon as feasible

## Disclosure

We follow responsible disclosure practices. We will coordinate with
reporters before making details public.
